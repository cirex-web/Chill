Blocked Sites:

What is the purpose of this extension?

This extension is meant to be used to help with self control and increase productivity. It can also be used to block certain pop-up websites/ads.

How can I block a site?

First, click on the “Block this site” button above. The button will lead you to a block settings page where you can block a website with a different URL than the one you’re currently on. If you don’t add any text to the input field, the extension will block the current webpage.

How can I temporarily unblock a site?

When you go to a blocked site, the webpage will be replaced with the default blocked site page. From there, you can request access to the site from your future self by entering the reason you want to access the site and how long you should wait until making the decision to unblock the site.

How can I permanently unblock a site?

You can’t permanently unblock a site that you block.
